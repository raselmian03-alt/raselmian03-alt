### About me

I'm a data analyst based in Sweden, passionate about finding patterns in data and building tools that make information easy to understand. I work mostly with Python — pandas, Streamlit, matplotlib — and I enjoy building interactive dashboards that tell a clear story.

- Working on trade data analysis between Sweden and Bangladesh
- Interested in economics, supply chains, and open government data
- Always looking for the story behind the numbers

### Featured project

<div align="center">

[![Sweden Bangladesh Trade](https://github-readme-stats.vercel.app/api/pin/?username=raselmian03-alt&repo=Sweden_Bangladesh_Trade_in_goods&theme=tokyonight&hide_border=true)](https://github.com/raselmian03-alt/Sweden_Bangladesh_Trade_in_goods)
